# Wind Direction

Converts wind direction from degrees to cardinal points


### Install

From npm:

```
npm install @marzzelo/windir
````


## Usage

```
const { windir } = require('./windir');
console.log(`dir: ${windir('270').direction}`);

// dir: Oeste  (West)
```
