# Wind Direction

Converts wind direction from degrees to cardinal points


### Install

From npm:

```
npm install marzzelo/windir
````


## Usage

```js
const { windir } = require('@marzzelo/windir')
console.log(`dir: %o`, windir(270))

// dir: { index: 12, direction: 'Oeste' }  (West)
```
